# PWA Icons

Place `icon-192.png` (192×192) and `icon-512.png` (512×512) here before deploying.

Quick option: generate them from an emoji using https://realfavicongenerator.net or https://favicon.io/emoji-favicons/ (use the ✉️ emoji).

Both files are referenced by `manifest.json` and the `<link rel="apple-touch-icon">` tag.
